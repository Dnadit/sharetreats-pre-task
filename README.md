## 폴더구조
- 각 폴더의 숫자는 사전과제 2번째 3번째라는 뜻입니다.

## 실행환경
- JDK 17

## 실행
- 2번과제 실행
  ```bash
  $ java -jar ./out/artifacts/sharetreats_pretask_2_jar/sharetreats-pretask-2.jar
  ```

- 3번과제 실행
  ```bash
  $ java -jar ./out/artifacts/sharetreats_pretask_jar/sharetreats_pretask.jar
  ```
